Design Tool UI

This page outlines the current design for the tool interface. The goal is to make the workflow readable, guide users through different fabrication modes, and connect free design, creative tools, and case reproduction in one consistent interface.

Some detailed UI drafts are still only available in Feishu and cannot be shown outside that platform. Original site images are used here as placeholders.

Framework

The interface is organized around three main parts: a free-design module, a creative tool module, and a case-reproduction module. Shared elements such as guidance diagrams, process notes, icons, and export actions remain visible throughout the interface.

Free Design Module

The free-design module begins with mode selection. Four modes are available, followed by a choice of cutting strategy such as parallel paths or crossed paths. The lower part of the interface changes depending on the selected mode and provides the relevant guidance diagram, process notes, and export function.

Key Elements

  • Select one of four design modes.
  • Select a cutting strategy: parallel or crossed.
  • Switch the lower panel based on the current workflow.
  • Show guidance diagrams and caution notes.
  • Export the result.

Creative Tool

The creative tool module supports several lightweight generators intended for playful or structured outputs.

Letter

The letter tool includes letter selection, size adjustment, and generation.

Storage Box

The storage-box tool is currently marked as a future item.

Seaweed

The seaweed tool uses two sliders for x and y control, along with board-thickness options of 2, 3, 5, 8, and 10, followed by generation.

City

The city tool is currently marked as a future item.

Seed

The seed tool uses size as the main parameter before generation.

Case Reproduction

The case-reproduction section provides fixed-size presets and downloadable assets for reproducing representative examples. It also includes a diagram or a photographed reference for each example.

Images and Icons

The interface also needs a consistent set of pictures, icons, and guidance diagrams so that fabrication modes are understandable at a glance.

Guidance Diagrams

Single-sided Process

  1. Calibration
  2. Cutting

Double-sided Process

  1. Calibration
  2. Cut the top face
  3. Flip the sheet like turning a book page
  4. Rotate by 90 degrees
  5. Cut the back face

Vertical-face Process

The vertical-face process is conceptually very similar to the single-sided process, but it swaps horizontal deformation and vertical deformation, so the parameters need to be adjusted accordingly.

  1. Vertical clamping
  2. Calibration
  3. Cutting

Six-face Process

The six-face process begins by slicing different faces, arranging the slices, and unfolding them on the same plane. In the UI, the important point is to communicate that fixtures are used to change both positioning and clamping state.

The detailed workflow is:

  1. Calibration
  2. Cut the top face
  3. Flip the sheet like turning a book page
  4. Rotate by 90 degrees
  5. Cut the back face
  6. Vertical clamping
  7. Cut the four side faces
