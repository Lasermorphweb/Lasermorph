Customizable Game Peripherals

This case develops a bio-based game controller that combines compliant material behavior with modular component organization. The prototype improves hand comfort during extended use while also supporting component substitution for different game genres and interaction patterns. Rather than treating the controller as a fixed object, the project frames it as a configurable tactile platform that can be adjusted according to ergonomic preference and play style.

The case demonstrates a sustainable hardware logic in which ergonomic fit, playful customization, and flat-packed fabrication are integrated into a single interface family. Its value lies not only in comfort, but also in lifecycle thinking: modules can be fabricated, swapped, stored, and distributed more flexibly than conventional rigid consumer hardware. As a result, the case points toward a "plug-and-play" paradigm that is materially softer, logistically lighter, and conceptually more reusable.
