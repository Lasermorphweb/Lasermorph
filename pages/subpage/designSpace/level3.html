Level 3: Volumetric Shapes

Level 3: Volumetric Shapes introduces the creation of complex models, focusing on 3D structures that go beyond simple shapes and curvatures. At this stage, users are required to build their own models using their design software and then import those models into the LaserMorph design tool for further processing. This level allows for the creation of intricate and highly customized volumetric shapes, offering flexibility in design.

In Level 3, users can create a wide range of 3D forms, including complex and organic shapes, which can then be processed using LaserMorph's unique method of material manipulation. Some examples of models that can be created include animals like rabbits, objects like cars, and many other intricate designs. These examples serve as inspiration for users to experiment and apply the tool in their designs. We also provide access to several pre-made example models that users can modify and customize based on their specific needs.

Level 3: Volumetric Shapes

Process

  1. Create Your Own Model: Start by building a 3D model in your preferred modeling software. This can be any complex shape that you wish to work with, such as organic forms, machinery, or abstract structures. The model should be designed with the desired complexity and details in mind.
  2. Import the Model into the Design Tool: After completing your model, import the file into the LaserMorph design tool. Supported file formats typically include DXF, STL, and OBJ. The design tool will allow you to manipulate and modify the model for laser cutting.
  3. Adjust Model Parameters: Once the model is imported, adjust the parameters for the laser processing, such as cutting depth, speed, and the number of passes. These adjustments will help prepare the model for the laser cutting process and ensure optimal results.
  4. Test Using Example Models: If you're new to the process or looking for inspiration, you can refer to example models provided in the tool, such as the rabbit or car models. These models can be used as templates for creating your own designs or as practice for understanding the tool's capabilities.
  5. Process the Model with LaserMorph: After making adjustments, use the LaserMorph tool to process your model. The tool will generate the necessary cutting paths and prepare the model for laser engraving and activation. This allows you to see how the material will behave once cut and activated.
  6. Observe the Final Result: Once the model is processed and activated, observe how the material responds and transforms into the final 3D shape. This step demonstrates the effectiveness of composite curvatures and other techniques used to create intricate and dynamic shapes.
