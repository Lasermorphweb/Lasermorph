Horizontal Joining

Horizontal expansion involves creating interlocking, zigzag edges that allow different sections of the material to be joined horizontally. The design relies on puzzle-like interlocking patterns, which can be adjusted based on the desired size of the final expanded shape.

Horizontal Joining

Process

  1. Create Base Model: Design a model with interlocking zigzag edges along the connecting areas.
  2. Design Zigzag Edges: Ensure that each side of the model has a zigzag pattern that will interlock with adjacent parts.
  3. Import to Design Tool: Import the model into the LaserMorph design tool.
  4. Generate Cutting Paths: The design tool will generate cutting paths for both the main shape and the interlocking zigzag edges.
  5. Cutting Process: Use the laser cutter to cut the interlocking edges, ensuring the pieces will fit together accurately.
  6. Completion: After cutting, connect the sections to form a larger, expanded prototype.
