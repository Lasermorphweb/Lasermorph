Multimodal Haptic Instruments

This case translates musical structure into tactile interaction by using laser-coded stiffness and geometry as part of the interface language. Different keys carry distinct tactile identities, and pressing them triggers synchronized audio-visual feedback, creating a closed loop between touch, sound, and image. The project therefore allows the user to feel tonal difference before or together with hearing it, turning musical abstraction into embodied experience.

The case is valuable for music education, embodied interaction, and multisensory design because it makes abstract tonal relations physically graspable. Rather than learning scale only as notation or sound, users experience it as variation in resistance, deformation, and pressing gesture. This greatly broadens the educational and experiential possibilities of the interface, suggesting that lasermorph can encode conceptual distinctions into differentiated tactile behaviors.
