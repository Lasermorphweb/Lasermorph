Level 3: Volumetric Shapes

Level 3: Volumetric Shapes introduces the creation of complex models, focusing on 3D structures that go beyond simple shapes and curvatures. At this stage, users are required to build their own models using their design software and then import those models into the LaserMorph design tool for further processing.

Level 3: Volumetric Shapes Overview

Process Steps

  1. Create Your Own Model: Start by building a 3D model in your preferred modeling software. This can be any complex shape that you wish to work with.
  2. Import the Model into the Design Tool: After completing your model, import the file into the LaserMorph design tool. Supported file formats typically include DXF, STL, and OBJ.
  3. Adjust Model Parameters: Once the model is imported, adjust the parameters for the laser processing, such as cutting depth, speed, and the number of passes.
  4. Test Using Example Models: If you're new to the process or looking for inspiration, you can refer to example models provided in the tool, such as the rabbit or car models.
  5. Process the Model with LaserMorph: After making adjustments, use the LaserMorph tool to process your model.
  6. Observe the Final Result: Once the model is processed and activated, observe how the material responds and transforms into the final 3D shape.
