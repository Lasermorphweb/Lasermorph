Design Tool

Our design tool, built on Grasshopper and utilizing the HumanUI component, provides an intuitive, easy-to-use interface for users. The tool is divided into four major modules: Calibration, Free Design, Creative Tools, and Case Restoration. These modules offer different features that cater to various design needs, from setting up calibration parameters to creating custom models using parametric design.

Design Tool Interface

1. Calibration

The Calibration module allows users to set up their laser cutter and ensure that the material is cut accurately. Users can select the power of their machine (e.g., 20W or 10W), and the system will recommend cutting speed and power percentage.

  • Machine Power Selection: Choose the laser cutter's power (e.g., 20W, 10W).
  • Recommended Parameters: Based on the selected power, the system will provide recommended cutting speed and power percentage.
  • Calibration Model Export: The system generates a calibration model, which can be exported as DXF or G-code files.
  • Power Calculation: Users can input the machine's current power percentage and speed, and the system will calculate the thickness of each cut.
  • Final Calibration: Use the calibration model to perform test cuts and ensure the cutting depth is consistent and accurate.

2. Free Design

The Free Design module provides users the flexibility to import their own models and generate cutting paths. The module supports various cutting modes, including single-sided, double-sided, side-plane, and six-sided cutting.

  • Single-Sided Cutting: Import the model and generate cutting paths for the front side. Preview the model's dry and wet state effects.
  • Double-Sided Cutting: Split the model into the front and back parts. Generate cutting paths for both sides.
  • Side-Plane Cutting: Similar to single-sided cutting, but requires adjusting parameters for the vertical direction.
  • Six-Sided Cutting: Use the above methods to cut all six faces of the model.

3. Creative Tools

The Creative Tools module offers four parametric design cases that allow users to create custom models by adjusting various parameters. These cases include Letter Arrangement, Seaweed, City, and Expansion Module.

  • Letter Arrangement Case: Users can input the desired letters, numbers, or simple shapes and adjust their size, rotation, and spacing.
  • Seaweed Case: Adjust the number of seaweed strands in the X and Y directions, height, overall size, and other parameters.
  • City Case: Import a curve from a file, then define the maximum and minimum heights of city elements.
  • Expansion Module Case: Customize the zigzag pattern for horizontal expansion and adjust parameters for vertical expansion.

4. Case Restoration

The Case Restoration module includes a selection of four classic pre-designed models that users can reference or use directly in their projects: Car, Rabbit, Wrist Rest, and UIST Cube.

  • Car Model: Single-sided cutting. Preview the model in both dry and wet states.
  • Rabbit Model: Double-sided cutting. Generate cutting paths for both sides.
  • Wrist Rest Model: Single-sided cutting with ergonomic curve design.
  • UIST Cube Model: Six-sided cutting for a fully enclosed cube.
