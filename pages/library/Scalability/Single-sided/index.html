Single-sided

Single-sided carving is the most basic technique, where the design is cut on one side of the material only. This method is suitable for simpler designs and provides a straightforward approach to creating 3D shapes.

Single-sided Carving

Process

  1. User Imports Model: Import the model into the design software.
  2. Generate Front File: The design software will generate the cutting file for the front side. Ensure the bottom-left corner of the front shape is positioned 10mm away from the XY axis.
  3. Output Cutting Files: Export the cutting file in DXF or other supported formats.
  4. Cutting Process: Load the material into the laser cutter and cut the front using the generated file.
  5. Completion: After cutting, you will have a fully carved shape, ready for activation.
