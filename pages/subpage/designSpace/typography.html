Design Space / Resolution / Typography

Typography

Typography refers to the minimum text size and readability achievable through the laser cutting process. Early tests have shown that 1.0cm diameter text is ideal for display, while 0.75cm text can still be recognized, but any smaller text becomes unreadable. This sets the limits for text size when incorporating typography into laser-cut designs.

Typography

Process

  1. Choose the Font and Size: Decide on the font and size for text-based elements in the design.
  2. Create Sample Text: Start by designing sample text with a 1.0cm diameter for the most readable and clear display.
  3. Test for Smaller Text: Test smaller sizes (e.g., 0.75cm) to verify readability after cutting.
  4. Adjust Text Size: If the text is smaller than 0.75cm, it may become illegible, so adjustments should be made.
  5. Final Typography: Ensure that all text is legible, and that the size falls within the recommended range.
