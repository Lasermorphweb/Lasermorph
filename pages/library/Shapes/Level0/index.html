Level 0: Surface Slopes

The Level 0: Surface Slopes section of the Shapes design space is focused on the ability to create angled surfaces. This section allows for cutting various units with different slope angles, which can be adjusted according to the design requirements. These angled surfaces can then be arranged in patterns to form unique geometries.

Level 0: Surface Slopes Overview

Part 1: Creating a Single Surface Unit

  1. Create a Cube Model: Begin by building a basic cube model in your design software.
  2. Define the Slope Angle: Select one edge on the top face of the cube to define an angle for the slope.
  3. Cut the Slope: Using the defined angle, make a cut along the selected edge of the cube to create the sloped surface.
  4. Final Unit Model: After the cut is complete, the resulting shape will be a single surface unit with the desired slope, which can be used in the larger pattern design.

Part 2: Creating the Zigzag Pattern

  1. Define the Distance Between Units: Set a fixed distance for how far apart each sloped unit will be placed along the pattern.
  2. Design a Flat Rectangle: Start with a flat rectangular shape, where the cross-section is square, and the length and width are defined based on your pattern needs.
  3. Perform Repeated Cuts: At regular intervals along the rectangle, make angled cuts, each at a consistent depth. These cuts should be made to a certain depth, roughly halfway through the material.
  4. Final Zigzag Model: Once the cuts are made, the resulting shape will be a zigzag pattern, formed by the alternating angled cuts along the flat surface.
