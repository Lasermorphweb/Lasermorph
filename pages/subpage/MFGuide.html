Mechanism and Fabrication

This guide explains the current structure of the project: how the material works, how it is prepared and calibrated, how fabrication is organized, and how parameters connect to design-space exploration and applications.

Understand How It Works

Workflow

The process starts from a compressed cellulose sponge sheet, uses laser engraving to encode local depth, activates swelling with water, and then reads the resulting 3D form after drying.

Mechanism

The method depends on anisotropic swelling. Different engraving depths change the local expansion potential, so height and curvature can be programmed in a flat state before activation.

Advantages

The process is fast for prototype iteration, easy to transport in a flat state, biodegradable, and suitable for repeated wet-dry cycles.

Material Preparation

Material

Compressed cellulose sponge is used because it is inexpensive, easy to obtain, biodegradable, and capable of about 10x vertical swelling.

Environment

Repeatable humidity, sheet thickness, and fixing conditions are important for stable results.

Equipment

Machine power, motion settings, and material batch differences all affect the final depth and therefore the swelling result.

Fabrication

Pre-processing Preparation

Before fabrication, the project uses test coupons, line-spacing checks, and power-speed comparisons to build a usable local parameter table.

Calibration: Testing and Development

The standard test coupon combines text, tactile modules, structural connection tests, wall-thickness checks, slope tests, irregular-surface checks, and depth tests. Instead of fixed global parameters, users establish a personal parameter dictionary from measured results.

Laser Cutting

The main variables are power, speed, line spacing, and pass count. Current records use line spacing around 0.20 mm as a useful baseline for many tests, but each setup still requires calibration.

Activation

Soaking triggers swelling and drying locks the result. These two states also reveal the difference between wet softness and dry stiffness.

Laser Parameters and Fabrication Tips

Fabrication Parameter Text

Current notes focus on swelling ratio, shrinkage ratio, and the relation between engraving depth and final height. These results feed directly into the design tool and later application work.

Link to Application Examples

The calibrated knowledge is reused in rapid prototypes, packaging, tactile interfaces, and enhanced sponge products.
