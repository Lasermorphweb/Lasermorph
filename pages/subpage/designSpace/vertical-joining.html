Design Space / Scalability / Vertical Joining

Vertical Joining

Vertical expansion allows users to create modular, interlocking pieces with protruding and recessed features along the vertical surfaces. These features enable users to assemble larger structures by interlocking multiple parts vertically.

Vertical Joining

Process

  1. Create Base Model: Design a model with protruding and recessed features along the vertical surfaces.
  2. Design Protrusions and Recesses: Ensure that the protruding and recessed features are designed so that they can fit together easily.
  3. Import to Design Tool: Import the model into the LaserMorph design tool.
  4. Generate Cutting Paths: The design tool will generate the cutting paths for the protruding and recessed features.
  5. Cutting Process: Use the laser cutter to cut the vertical features along the model.
  6. Completion: After cutting, fit the pieces together to create a larger, vertically assembled structure.
